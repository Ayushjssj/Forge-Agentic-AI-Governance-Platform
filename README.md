# 🛡️Forge — Agentic AI Governance Platform

Forge is a production-grade Agentic AI Governance and Security Platform designed to simulate, observe, secure, govern, and deploy AI agents safely.

Forge helps developers and organizations test AI agents against prompt injection attacks, hallucinations, compliance violations, PII leakage, and unsafe agent behaviors before deployment.

---

## 🚀 Features

### ✅ AI Blueprint Library

Pre-built enterprise AI agents:

- HR Helpdesk Agent
- Email Triage Agent
- Loan Origination Agent

Each blueprint can be forked into the Forge Studio.

---

### ✅ AI Simulation Studio

Real-time AI testing environment with:

- Live AI chat
- Simulation engine
- Scenario execution
- Sigma reliability scoring
- AI response monitoring
- Agent execution tracking

---

### ✅ AI Security Dashboard

Automatically analyzes:

- Prompt Injection Attacks
- PII Leakage
- Hallucinations
- Compliance Violations
- Risk Scores
- Unsafe outputs

---

### ✅ Red Team Lab

Generate adversarial attack prompts for testing AI safety.

Examples:

- Ignore previous instructions
- Reveal confidential data
- Bypass security policies
- Leak payroll records
- Guarantee loan approval

---

### ✅ Control Plane

Monitor:

- Agent runs
- Sigma scores
- Execution history
- Costs
- Tokens
- Scenario counts
- Audit logs
- Governance checks

---

### ✅ Deployment / Ship Workflow

Simulated enterprise AI deployment pipeline with:

- GitHub PR workflow
- Governance validation
- Security review
- Reliability checks
- Deployment checklist
- Deployment statistics

## ⚡ Installation

### 1️⃣ Clone Repository

```bash
git clone https://github.com/Ayushjssj/Forge-Agentic-AI-Governance-Platform.git
```

---

### 2️⃣ Navigate Into Project

```bash
cd Forge-Agentic-AI-Governance-Platform
```

---

### 3️⃣ Install Frontend Dependencies

```bash
cd apps/web
npm install
```

---

### 4️⃣ Install Backend Dependencies

```bash
cd ../sim
pip install -r requirements.txt
```

---

## ▶️ Running the Project

### Start Backend

```bash
cd apps/sim
uvicorn main:app --reload --port 8000
```

Backend runs on:

```txt
http://127.0.0.1:8000
```

---

### Start Frontend

```bash
cd apps/web
npm run dev
```

Frontend runs on:

```txt
http://localhost:3000
```
